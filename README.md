# NoxChat
This singular script file is just 7kb in size and can be drag-and-dropped into any figura model for free and should work immediately both on and off of the SMP. Pictured above, it is a condensed script that formats your chat with **custom colors, bold, italics, underlines, strikethrough, and even spoilering/obfuscation.** ***With 100% Discord parity***.
__How do I install it__?
- simply add this file to your personal avatar folder (not "avatars," but the avatar you are currently wearing).
- If you are a dev who has disabled autoscripts, be sure to also add `require("noxchat")` to your script.
- if you have issues, request help
__How do I use it__?
- using the & symbol, you can type any minecraft formatting code. look it up if you don't know, or reference the below gif.
- alternatively, use these: \**italic*\*, \*\***bold**\*\*, \_\___underline__\_\_, \~\~~~strikethrough~~\~\~, \|\|||spoiler/obfuscate||\|\|
- hovering over spoilered/obfuscated text shows you what it says!
- want a custom color? use a hex code! with this format `&(ffffff)` or `&(fff)` you can apply custom colors to your chat. Note that doing this will show up in the discord chat though, whereas regular `&7` text codes don't.
- want to automatically format your messages? Say a specific character speaks an unknown language, or speaks only in red. check the top of the script for a setup config that gives you auto-formatting as well as a way to "cancel" auto-formatting in a message.
- want to speak a certain font? by default, use `f:(fontname)` to substitute your messages. See the top of the script for more info.
- want to play a sound when you speak? check the top of the script for setup.
- note that the above three features will require Dev > Chat Messages: ON in your Figura settings.
